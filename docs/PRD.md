# Project Predicte PRD

Project Predicte replaces spreadsheet-based football prediction leagues with a mobile app for private leagues.

Milestone 0 delivers only a verified mock vertical slice:

- browse a mock World Cup-style competition;
- create or join a mock private league;
- enter group-stage predictions quickly;
- simulate saved/synced prediction state;
- settle a mock official result;
- generate scoring events and a leaderboard snapshot;
- show rank movement.

Excluded from this milestone: payments, entry fees, prize pools, paid member tracking, advertising, betting, odds, gambling, real sports-provider APIs, and real Supabase Auth.
